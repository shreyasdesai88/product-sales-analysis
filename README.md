# product-sales-analysis
This is  Flexible data comparison and analytics dashboard built in Python using Streamlit and scikitlearn.It is a dynamic system for analysis of any dataset in realtime. It automatically understand structure, detect the main comparison entity, and generate analysis, charts, trends, and insights without changing the source code for each new dataset. It has been designed as a generic analytics system for any formof dataset to give insights
Expanding this to ML Project will create product optimization for maximum sales. It will analyze previous sales and provide insights for maximum profit. Used scikitlearn for ML and AI based optimized sales price.it will help businesses and companies increase sales. ML part will show where company needs to improve in sales. It will support businesses grow and generate sales creating profits.
